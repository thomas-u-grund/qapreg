# Print a QAP regression object

Prints coefficient estimates, model-based standard errors, QAP standard
errors, and QAP p-values for a fitted `qapreg` object.

## Usage

``` r
# S3 method for class 'qapreg'
print(x, digits = 4, ...)
```

## Arguments

- x:

  An object of class `"qapreg"`.

- digits:

  Number of digits used for rounding printed output.

- ...:

  Additional arguments, currently ignored.

## Value

Invisibly returns `x`.
